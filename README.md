# Geodata ETL Pipeline (Python + PostGIS)

## Overview
This project implements a geospatial ETL workflow using Python and PostGIS.

## Features
- Data extraction
- Attribute transformation
- Spatial quality checks
- SQL processing pipeline
- Dockerized PostGIS database

## Tech Stack
Python, PostgreSQL/PostGIS, SQL, Docker

## How to run
docker-compose up -d
python src/run_pipeline.py
